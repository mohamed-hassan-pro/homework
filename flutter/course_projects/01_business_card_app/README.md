## 📖 Project Overview
The Business Card App is a minimalistic, elegant digital business card application built to display professional contact information. It demonstrates clear usage of Flutter's foundational layout widgets to present a portrait profile, name, title, and contact avenues effectively.

## ✨ Key Features
*   **Profile Presentation:** Cleanly displays a prominent circular avatar emphasizing the physical profile.
*   **Contact Information Cards:** Reusable user interfaces to represent contact details like phone number and email with distinct icons.
*   **Responsive Layout Layout:** Fully centered structure designed to scale cleanly using core layout positioning.

## 🧠 Lessons Learned
*   **Fundamental UI Structures:** Constructed stable interfaces relying heavily on fundamental layout widgets such as `Column`, `Container`, `Padding`, and `Divider`.
*   **Widget Refactoring:** Gained proficiency in promoting code reusability by extracting repetitive structures into custom components (`InputSection`, `AvatarSection`).
*   **Theming and Assets:** Applied custom stylistic fonts (Pacifico) and appropriately loaded local image assets to polish the final visual aesthetic.
